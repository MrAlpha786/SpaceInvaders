# SpaceInvaders
Space Invader game written in Python 

### created by [MrAlpha786](https://github.com/MrAlpha786)

# Requirements

* Python 3.*
* Pygame

# Controls

key    | Action
:-----:|:--------------:
left   |  move left
right  |  move right
space  |  fire bullet
q      |  quit
ESC    |  pause/play
